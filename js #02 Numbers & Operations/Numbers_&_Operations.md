# Numbers & Operations

 
## 1. Операції

```javascript
let addition = 10 + 5; // додавання // 15
let subtraction = 10 - 5; // віднімання // 5
let multiplication = 10 * 5; // множення // 50
let division = 10 / 5; // поділ // 2
let exponentiation = 10 ** 5; // піднесення до ступеня // 100000
let remainder = 17 % 7; // остаток від поділу // 3
```

#### *Пріоритет:*

```javascript
console.log(5 + 1 * 10); // 15
console.log((5 + 1) * 10); // 60
```

## 3. Числові літерали

- Число **123** в різних системах числення:
```javascript
let dec = 123; // 10-кова система
let hex = 0x7b; // `0x` - префікс 16-кова системи. Саме число - `7b`
let bin = 0b1111011; // `0b` - префікс 2-ковій системи. Саме число - `1111011`
let oct = 0o173; // `0o` - префікс 8-кова системи. Саме число - `173`
```

- Різні варіації десятичних чисел:
```javascript
let x = 15; // ціле десятичне
let y = 0.52; // десятичне с дробовою частиною
let z = 3.14e6; // `e6` означає помножити на `10` в `6` ступені, тобто наше число - `3140000`
```

## 4. Десяткова, двійкова та шістнадцяткова системи

- Звична нам **десяткова система** позиційна, тому кожна цифра множиться на 10 відповідно (позиція від правого краю, рахуючи з 0). Розглянемо число 345:
```javascript
345 === 3 * (10 ** 2) + 4 * (10 ** 1) + 5 * (10 ** 0)
345 === (3 * 100) + (4 * 10) + (5 * 1)
```

- У **двійковій системі** є лише числа 0 і 1, які потрібно множити на 2 відповідно:
```javascript
0b101101 === 1 * (2 ** 5) + 0 * (2 ** 4) + 1 * (2 ** 3) + 1 * (2 ** 2) + 0 * (2 ** 1) + 1 * (2 ** 0)
0b101101 === (1 * 32) + (0 * 16) + (1 * 8) + (1 * 4) + (0 * 2) + (1 * 1)
0b101101 === 32 + 8 + 4 + 1 === 45
```

- У **шістнадцятковій системі** використовуються числа від 0 до 9 та літери A = 10, B = 11, C = 12, D = 13, E = 14 и F = 15:
```javascript
0x9F2C = 9 * (16 ** 3) + 15 * (16 ** 2) + 2 * (16 ** 1) + 12 * (16 ** 0)
0x9F2C = (9 * 4096) + (15 * 256) + (2 * 16) + (12 * 1)
0x9F2C = 36864 + 3840 + 32 + 12 === 40748
```

## 5. Округлення значень

- `Math.round(x)` - округлення до найближчого цілого (вгору чи вниз).
```javascript
Math.round(10.49); // 10
```

- `Math.floor(x)` - округлення ВНИЗ до найближчого цілого.
```javascript
Math.floor(10.51); // 10
```

- `Math.ceil(x)` - округлення ВГОРУ до найближчого цілого.
```javascript
Math.ceil(10.49); // 11
```

- `Math.trunc(x)` - відкидає дробову частину.
```javascript
Math.trunc(10.342); // 10
```

- `x.toFixed(numberOfDigits)` - перетворює число в рядок із заданою кількістю цифр після коми.
```javascript
(3.14159).toFixed(3); // '3.142'
```

## 6. Перетворення Рядка в Число

- `+` - для суворого перетворення
```javascript
const text = '3.14159';
console.log(
  +text, // 3.14159
  +'27.5', // 27.5
  +'2.4e3', // 2400 - експоненціальний формат
  +'0xABC', // 2748 - шестнадцяткове число
  +'0b1001', // 9 - бінарне число
);
```

- `Number` - аналогічно `+`
```javascript
console.log(
  Number(text), // 3.14159
  Number('27.5'), // 27.5
  Number('2.4e3'), // 2400 - експоненціальний формат
  Number('0xABC'), // 2748 - шестнадцяткове число
  Number('0b1001'), // 9 - бінарне число
);
```

- Перетворення інших типів:
```javascript
console.log(+true); // 1
console.log(+false); // 0
console.log(+null); // 0
console.log(+undefined); // NaN
```

#### *Для часткового перетворення числового рядка використовуються стандартні функції `parseInt` та `parseFloat`*:
- `parseInt` приймає два параметри: значення для перетворення та систему обчислення (опціонально), і повертає ціле число, або `NaN`, якщо рядок не починається з числа:
```javascript
parseInt('256$'); // 256 – зупиняємось на першому символі, який не є частиною цілого числа
parseInt('123.456'); // 123 – дробова частина відкидається
parseInt('256', 2); // NaN – бо такі числа не існують у 2-й системі
parseInt('1A'); // 1 – буква `А` ігнорується
parseInt('1A', 16); // 26 - шістнадцяткове число
parseInt('abc'); // NaN – рядок не починається з числа у 10-й системі
```

- `parseFloat` приймає значення для перетворення і число або `NaN`, якщо конвертація неможлива, наприклад:
```javascript
parseFloat('3.14'); // 3.14
parseFloat('2.718 is the base of natural logarithm'); // 2.718
parseFloat('abc'); // NaN, тому що рядок не починається з числа
```

## 7. Infifnity & NaN


У JavaScript існує 2 спеціальних числових значення: **Infinity** і **NaN**.

- `Infinity` – це математична нескінченність. Вона виникає при розподілі на 0 і більше від будь-якого числа. Також є `-Infinity` - вона менша за будь-яке число.

- `isFinite()` - функція, яка перетворює аргумент на число і повертає `true`, якщо це звичайне число, а не `NaN` або `Infinity`.

- `NaN` – це спеціальне числове значення, що виникає при помилці обчислення. Наприклад, якщо 0 розділити на 0, або число на рядок, яка не може бути наведена до `(346 / 'apple')`. Значення `NaN` унікальне тим, що воно не дорівнює нічому іншому, навіть самому собі:
```javascript
console.log(NaN === NaN); // false
```

- `isNaN` - функція, яка перетворює свій аргумент на число і визначає, чи є воно `NaN` (не числом) чи ні:
```javascript
isNaN(NaN); // true
isNaN(45.2); // false
isNaN(Infinity); // false
isNaN('123'); // false
isNaN('0x123'); // false
isNaN('12 hello'); // true
```

## 8. Math.random()

- `Math.random()` - метод, який повертає випадкове число від 0 (включно) до 1 (не включно):
```javascript
console.log(
  Math.random(), // 0.6460395007021751
  Math.random(), // 0.511311069945108
  Math.random(), // 0.36258333704200685
);
```

- Для генерації випадкових цілих чисел від `0` до `n` слід помножити результат на `n + 1` і округлити вниз (до меншого цілого):
```javascript
Math.floor(Math.random()*11); // випадкове ціле число від 0 до 10 включно
```

- Якщо необхідно випадкове ціле число від `min` до `max` включно, то генерують число від `0` до `max - min` і додають `min`:
```javascript
Math.floor(Math.random()*(max - min + 1)) + min;
Math.floor(Math.random()*(11 - 7 + 1)) + 7; // випадкове ціле число від 7 до 11 включно
Math.floor(Math.random()*(20 - 10 + 1)) + 10; // випадкове ціле число від 10 до 20 включно
```
